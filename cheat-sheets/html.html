    Common Elements

    Block container: div

    Inline container: span


    Horizontal rule: hr

    Reture: br


    Specify bold: b

    Specify italics: i


    Good alternatives:

    Bold: strong

    Italics: em


    Specify typewriter-like font: tt

    Increase the size of the font: big

    Decrease the size of the font: small

    Specify a subscript: sub

    Specify a superscript: sup

    Dotted underline: abbr


    Unordered List: ul

    Ordered ListL ol

    List item: li


    Description List: dl

    Define a term or name in the list: dt

    Describe the term or name: dd


    Image: img

    Link: a



    Tables

    Base: table

    Table header: thead

    Table body: tbody

    Table footer: tfoot

    Row: tr

    Head cell: th

    Cell: td


    Spans two columns (used in th or td): colspan="2"

    Spans two rows (used in th or td): rowspan="2"



    Forms

    Base: form

    With the form element: method, action, enctype


    Lables the inputs: lable


    Common input types:

    text, password, radio, checkbox, file


    Select list: select

    Options in the select list: option


    Multiple-line input box: textarea


    Button: button

    Button types: submit, reset


    Groups the inputs: fieldset

    Titles the group: legend



    Form Validation

    Common attributes:

    required, pattern, min, max

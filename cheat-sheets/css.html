    Use CSS in a HTML Document

    Internal: style="color: blue;"

    Embedded: style in head

    External: link

    By importing: @import url("/css/style.css");



    Selectors

    General: only element name

    Class: .class

    Only p elements with class "mytd": p.mytd

    ID: #id

    Universal: *

    Pseudo: a:hover, a:link, a:visited, p:hover

    Attribute: a[title], a[href *= "google"], a[href = "https://google.com"]

    Descendant: ul li a

    Adjacent: h1 + p

    Child: ul > li

    :nth-child(): li:nth-child(2), li:nth-child(4n), li:nth-child(even)

    Colour

    RGB and RGBA functions

    rgb(255, 100%, 0), rgba(0, 255, 100%, 0.5)

    Hex: #ff00ff

    Other colour functions: hsl(), hsla()



    Background

    Colour: background-color: #767392;

    Image: background-image: url("images/image.png");

    Repeat: background-repeat: no-repeat;

    Attachment: background-attachment: fixed;

    Position: background-position: 20px 20px;

    In a single line: background: #767392 url("image.png") no-repeat fixed 20px 20px;



    Box Model

    Content - The content of the box, which includes text and images.

    Padding - Padding refers to the transparent space between the element’s content and its border.

    The background colour of an element will also be seen through its padding area if it has one.

    Border - The padding and content are surrounded by a border.

    Margin - Margin refers to the transparent area around the border. The margin area is always transparent;

    it is unaffected by the element’s background colour, but it allows the parent element’s background color to be seen through it.



    Variables

    Declare: :root { --myBox: 100px; --myBoxColour: red; }

    Use: background-color: var(--myBoxColour);

    Calculate: width: calc(var(--myBox) * 2);

    Display

    Block-level Elements

    p, h1 ~ h6, ul, ol, hr, div, table, fieldset, form

    Inline elements

    a, b, i, u, em, strong, sup, sub, big, small, span, img


    Display: display:flex;

    Values: block, inline, inline-block, none, flex

    Flex: flex-direction, align-items, justify-content



    Visibility

    The element is hidden, but it still takes up space in the layout when you use visibility: hidden;

    hides the element, but it still takes up space in the layout.

    If the visibility of a hidden box’s child element is set to visible, it will be visible.

    On the other hand, display: none; disables the element’s display and removes it entirely from the page.

    visibility:hidden;
